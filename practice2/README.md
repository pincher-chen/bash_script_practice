# To generate Gaussian input files

```
# step 1, download ligand coordinates from http://quantum-machine.org/datasets/. Here we use qm8 dataset.
$ wget http://deepchem.io.s3-website-us-west-1.amazonaws.com/datasets/gdb8.tar.gz

# 使用babel软件对sdf文件进行分割
babel qm8.sdf -osdf qm8-.sd -m

# 提取sd文件的坐标
```
